# moredakka contract

## Inputs
- cwd
- git root
- branch
- status
- changed files
- diff
- recent commits
- nearby docs

## Roles
- planner
- implementer
- breaker
- minimalist
- synthesizer

## Outputs
- inferred objective
- selected path
- next actions
- commit plan
- tests
- edit targets
- risks
- disagreements
- stop conditions
- confidence

## Biases
- local context before broad speculation
- diff over file dump
- bounded loops
- one recommended path
- reversible moves over sweeping ones
